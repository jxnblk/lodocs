Flex Object
\n
CSS layout module using flexbox
\njxnblk.com/flex-object
\n
The flex-object is a powerful suite of utilities based on the\nflexible box layout module.\nThese utilities can replace the need for a grid system in many instances,\nbut can also be combined with the Basscss grid for even more possibilities.
\n
Note: browser support for the flexible box layout module is limited. See http://caniuse.com/#feat=flexbox
\n
.flex
\n
To begin, add the .flex class to a parent element.\nThis sets the element to display: flex and enables a flex context for all direct child elements.
\n
<div class="flex">\n  <a href="#" class="button">Link</a>\n  <a href="#" class="button">Link</a>\n</div>\n
\n
.flex-wrap
\n
To wrap elements, use the .flex-wrap utility.
\n
<div class="flex flex-wrap">\n  <a href="#" class="button">Home</a>\n  <a href="#" class="button">Nav Item</a>\n  <a href="#" class="button">Nav Item</a>\n  <a href="#" class="button">Nav Item</a>\n  <a href="#" class="button">Nav Item</a>\n  <a href="#" class="button">Nav Item</a>\n  <a href="#" class="button">Nav Item</a>\n  <a href="#" class="button">Nav Item</a>\n</div>\n
\n
.flex-column
\n
To arrange items vertically, use the .flex-column utility.
\n
<div class="flex flex-column">\n  <div class="p1 blue border">Row</div>\n  <div class="p1 blue border">Row</div>\n  <div class="p1 blue border">Row</div>\n</div>\n
\n
Alignment
\n
Use these utilities to control vertical alignment of child elements.
\n
.flex-center
\n
To vertically center child elements, add the .flex-center class to the parent element.
\n
<div class="flex flex-center">\n  <div class="flex-auto px2 py4 blue border">\n    <h1 class="m0">Big</h1>\n  </div>\n  <div class="px2 blue border">Small</div>\n</div>\n
\n
.flex-stretch
\n
To stretch all elements to fill the height of the parent, use the .flex-stretch utility.
\n
<div class="flex flex-stretch">\n  <div class="flex-auto px2 py4 blue border">\n    <h1 class="m0">Big</h1>\n  </div>\n  <div class="px2 blue border">Small</div>\n</div>\n
\n
.flex-stretch with nested content
\n
To ensure nested elements stretch as well, add the .flex class to relevant elements.
\n
<div class="flex flex-stretch">\n  <div class="flex-auto px2 py4 blue border">\n    <h1 class="m0">Big</h1>\n  </div>\n  <div class="flex blue border">\n    <div class="px2 white bg-blue">\n      Small\n    </div>\n  </div>\n</div>\n
\n
.flex-baseline
\n
To align child elements to their baseline, use the .flex-baseline utility.
\n
<div class="flex flex-baseline blue border-bottom">\n  <div class="flex-auto px2">\n    <h1 class="m0">Big</h1>\n  </div>\n  <div class="px2">Small</div>\n</div>\n
\n
.flex-start
\n
To align child elements to the top, use the .flex-start utility.
\n
<div class="flex flex-start blue border-top">\n  <div class="flex-auto px2">\n    <h1 class="m0">Big</h1>\n  </div>\n  <div class="px2">Small</div>\n</div>\n
\n
.flex-end
\n
To align child elements to the bottom, use the .flex-end utility.
\n
<div class="flex flex-end blue border-bottom">\n  <div class="flex-auto px2">\n    <h1 class="m0">Big</h1>\n  </div>\n  <div class="px2">Small</div>\n</div>\n
\n
Responsive Utilities
\n
To set display flex only at certain breakpoints and up,\nuse the breakpoint-prefixed versions of .flex.\nAll other utilities will only work at the set breakpoint.
\n
<div class="sm-flex flex-center mb2 blue">\n  <div class="border">\n    <h2 class="m0">.sm-flex</h2>\n  </div>\n  <div class="flex-auto border">Div</div>\n  <div class="border">Div</div>\n</div>\n<div class="md-flex flex-center mb2 blue">\n  <div class="border">\n    <h2 class="m0">.md-flex</h2>\n  </div>\n  <div class="flex-auto border">Div</div>\n  <div class="border">Div</div>\n</div>\n<div class="lg-flex flex-center mb2 blue">\n  <div class="border">\n    <h2 class="m0">.lg-flex</h2>\n  </div>\n  <div class="flex-auto border">Div</div>\n  <div class="border">Div</div>\n</div>\n
\n
Child elements
\n
To control the size and behavior of child elements, use these utilities.
\n
.flex-auto
\n
To make an element grow or shrink as needed, use the .flex-auto utility.
\n
<div class="flex">\n  <a href="#" class="button button-nav-light">Nav Item</a>\n  <a href="#" class="button button-nav-light">Nav Item</a>\n  <div class="flex-auto blue border"></div>\n  <a href="#" class="button button-nav-light">Nav Item</a>\n</div>\n
